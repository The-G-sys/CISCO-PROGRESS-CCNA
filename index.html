🌐 1. Network Fundamentals - Commandes de base ▼
! Modes de commande
enable ! Passe en mode privilégié
configure terminal ! Mode configuration globale
exit ! Retour au mode précédent
end ! Retour direct au mode privilégié
disable ! Retour au mode utilisateur
! Sauvegarde et gestion
copy running-config startup-config ! Sauvegarde la config
write memory ! Alternative pour sauvegarder
show running-config ! Affiche la config active
show startup-config ! Affiche la config sauvegardée
show version ! Version IOS et infos système
reload ! Redémarre l'équipement
! Diagnostic réseau
ping <ip> ! Test de connectivité ICMP
traceroute <ip> ! Trace le chemin des paquets
show ip interface brief ! Résumé des interfaces
show interfaces ! Détails des interfaces
show cdp neighbors ! Voisins CDP
show cdp neighbors detail ! Détails voisins CDP
show lldp neighbors ! Voisins LLDP
show lldp neighbors detail ! Détails voisins LLDP
! Configuration de base
hostname <nom> ! Nom de l'équipement
enable secret <mot_de_passe> ! Mot de passe privilégié
service password-encryption ! Chiffre les mots de passe
banner motd #message# ! Message d'accueil
line console 0 ! Configuration console
line vty 0 4 ! Configuration VTY (Telnet/SSH)
password <mot_de_passe> ! Mot de passe ligne
login ! Active la demande de connexion
! SSH (accès sécurisé)
ip domain-name cisco.com ! Nom de domaine (requis pour SSH)
crypto key generate rsa modulus 2048 ! Génère les clés RSA
username admin secret <mdp> ! Crée un utilisateur
transport input ssh ! Autorise uniquement SSH
login local ! Utilise la base locale
🔌 2. Network Access - VLAN, STP, EtherChannel ▼
! VLANs
vlan 10 ! Crée le VLAN 10
name SALES ! Nomme le VLAN
interface vlan 10 ! Interface virtuelle VLAN
ip address 192.168.10.1 255.255.255.0 ! IP du VLAN
no shutdown ! Active l'interface
! Ports d'accès
interface GigabitEthernet0/1 ! Sélectionne l'interface
switchport mode access ! Mode accès
switchport access vlan 10 ! Assigne au VLAN 10
switchport nonegotiate ! Désactive DTP
! Trunking 802.1Q
interface GigabitEthernet0/24 ! Interface trunk
switchport mode trunk ! Mode trunk
switchport trunk native vlan 99 ! VLAN natif
switchport trunk allowed vlan 10,20,30 ! VLANs autorisés
switchport trunk allowed vlan add 40 ! Ajoute un VLAN
! Spanning Tree Protocol (STP)
spanning-tree mode rapid-pvst ! Active Rapid-PVST+
spanning-tree vlan 10 root primary ! Root bridge pour VLAN 10
spanning-tree vlan 10 root secondary ! Root bridge secondaire
interface Gi0/1
spanning-tree portfast ! PortFast (accès rapide)
spanning-tree bpduguard enable ! BPDU Guard
spanning-tree portfast disable ! Désactive PortFast
! EtherChannel
interface range Gi0/1-2 ! Sélectionne les interfaces
channel-group 1 mode active ! LACP actif
channel-group 1 mode desirable ! PAgP desirable
interface Port-channel 1 ! Interface Port-channel
switchport mode trunk ! Configuration du Port-channel
! Routage Inter-VLAN (Router-on-a-Stick)
interface GigabitEthernet0/0.10 ! Sous-interface
encapsulation dot1Q 10 ! Encapsulation 802.1Q VLAN 10
ip address 192.168.10.1 255.255.255.0 ! IP de la sous-interface
! Commandes de vérification
show vlan brief ! Résumé des VLANs
show vlan id 10 ! Détails du VLAN 10
show interfaces trunk ! Interfaces trunk
show spanning-tree ! État STP
show spanning-tree vlan 10 ! STP pour VLAN 10
show etherchannel summary ! Résumé EtherChannel
show interfaces Gi0/1 switchport ! Infos switchport
🛣️ 3. IP Connectivity - Routage et OSPF ▼
! Routage statique
ip route 192.168.2.0 255.255.255.0 10.0.0.2 ! Route statique
ip route 0.0.0.0 0.0.0.0 10.0.0.1 ! Route par défaut
ipv6 route 2001:db8::/32 2001:db8:1::1 ! Route IPv6
ip route 192.168.2.0 255.255.255.0 Gi0/0 ! Via interface
! OSPFv2 (Open Shortest Path First)
router ospf 1 ! Active OSPF processus 1
router-id 1.1.1.1 ! Router-ID manuel
network 192.168.1.0 0.0.0.255 area 0 ! Annonce réseau
network 10.0.0.0 0.0.0.3 area 0 ! Avec wildcard mask
passive-interface Gi0/0 ! Interface passive
passive-interface default ! Toutes interfaces passives
no passive-interface Gi0/1 ! Sauf Gi0/1
default-information originate ! Annonce route par défaut
! OSPF multi-zone
network 192.168.1.0 0.0.0.255 area 1 ! Zone 1
network 10.0.0.0 0.0.0.3 area 0 ! Zone 0 (backbone)
area 1 range 192.168.0.0 255.255.0.0 ! Résumé de zone
! FHRP - HSRP (Hot Standby Router Protocol)
interface Gi0/0
standby 1 ip 192.168.1.254 ! IP virtuelle HSRP
standby 1 priority 110 ! Priorité (défaut 100)
standby 1 preempt ! Préemption activée
standby 1 authentication md5 key-string Cisco123 ! Authentification
standby 1 track Gi0/1 20 ! Tracking d'interface
! Commandes de vérification
show ip route ! Table de routage
show ip route ospf ! Routes OSPF uniquement
show ip route static ! Routes statiques
show ip protocols ! Protocoles de routage actifs
show ip ospf neighbor ! Adjacences OSPF
show ip ospf interface ! Interfaces OSPF
show ip ospf database ! Base de données OSPF
show standby ! État HSRP
show standby brief ! Résumé HSRP
⚙️ 4. IP Services - DHCP, NAT, NTP ▼
! DHCP Server
ip dhcp excluded-address 192.168.1.1 192.168.1.10 ! Exclusions
ip dhcp pool LAN_POOL ! Crée le pool DHCP
network 192.168.1.0 255.255.255.0 ! Réseau du pool
default-router 192.168.1.1 ! Passerelle par défaut
dns-server 8.8.8.8 ! Serveur DNS
domain-name monentreprise.local ! Nom de domaine
lease 7 ! Durée du bail (jours)
! DHCP Relay
interface Vlan 10
ip helper-address 192.168.1.100 ! Redirige DHCP vers serveur
! NAT/PAT (Network Address Translation)
access-list 1 permit 192.168.1.0 0.0.0.255 ! ACL pour NAT
ip nat inside source list 1 interface Gi0/0 overload ! PAT
interface Gi0/1
ip nat inside ! Interface interne
interface Gi0/0
ip nat outside ! Interface externe
! NTP (Network Time Protocol)
ntp server 192.168.1.100 ! Serveur NTP
ntp master ! Ce routeur est maître NTP
clock timezone CET 1 ! Fuseau horaire
clock summer-time CEST recurring ! Heure d'été
! SNMP et Syslog
snmp-server community MONSECRET ro ! Communauté SNMP lecture seule
snmp-server community MONSECRET rw ! Communauté SNMP lecture/écriture
logging 192.168.1.100 ! Serveur Syslog
logging trap informational ! Niveau de logs
! QoS (Quality of Service) - Bases
mls qos ! Active QoS sur switch
interface Gi0/1
mls qos trust cos ! Fait confiance au CoS
! Commandes de vérification
show ip dhcp binding ! Baux DHCP actifs
show ip dhcp pool ! Pools DHCP
show ip nat translations ! Traductions NAT actives
show ip nat statistics ! Statistiques NAT
show ntp status ! État NTP
show ntp associations ! Associations NTP
show clock ! Heure actuelle
show snmp ! Configuration SNMP
🔒 5. Security Fundamentals - ACL, Port Security, AAA ▼
! ACL Standard (1-99)
access-list 10 permit 192.168.1.0 0.0.0.255 ! Autorise réseau
access-list 10 deny any ! Refuse tout le reste
access-list 10 permit host 192.168.1.100 ! Hôte spécifique
interface Gi0/0
ip access-group 10 in ! Applique ACL en entrée
ip access-group 10 out ! Applique ACL en sortie
! ACL Étendue (100-199)
access-list 100 permit tcp 192.168.1.0 0.0.0.255 host 10.0.0.1 eq 80 ! HTTP
access-list 100 permit tcp any host 10.0.0.1 eq 443 ! HTTPS
access-list 100 deny ip any any ! Refuse tout
access-list 100 permit icmp any any ! Autorise ICMP (ping)
! Named ACL (ACL nommées)
ip access-list standard MY_ACL ! ACL standard nommée
permit 192.168.1.0 0.0.0.255
deny any
ip access-list extended WEB_ACL ! ACL étendue nommée
permit tcp any any eq 80
permit tcp any any eq 443
! Port Security
interface Gi0/1
switchport mode access ! Requis pour port security
switchport port-security ! Active port security
switchport port-security maximum 2 ! Max 2 adresses MAC
switchport port-security violation shutdown ! Action: shutdown
switchport port-security violation restrict ! Action: restrict
switchport port-security violation protect ! Action: protect
switchport port-security mac-address sticky ! Apprend dynamiquement
switchport port-security mac-address 0000.1111.2222 ! MAC statique
! AAA (Authentication, Authorization, Accounting)
aaa new-model ! Active AAA
aaa authentication login default local ! Auth locale par défaut
aaa authentication login CONSOLE line ! Auth pour console
radius-server host 192.168.1.100 key Cisco123 ! Serveur RADIUS
tacacs-server host 192.168.1.101 key Cisco123 ! Serveur TACACS+
aaa authentication login default group tacacs+ local ! TACACS+ puis local
! Sécurisation de base
enable secret <mot_de_passe> ! Mot de passe privilégié chiffré
service password-encryption ! Chiffre tous les mots de passe
no ip domain-lookup ! Désactive la résolution DNS
exec-timeout 5 30 ! Déconnexion après 5min30 d'inactivité
logging synchronous ! Synchronise les logs
! Commandes de vérification
show access-lists ! Toutes les ACL
show access-lists 100 ! ACL 100 spécifique
show ip interface Gi0/0 ! Infos interface avec ACL
show port-security ! Port security globale
show port-security interface Gi0/1 ! Port security interface
show port-security address ! Adresses MAC sécurisées
show aaa sessions ! Sessions AAA actives
show users ! Utilisateurs connectés
🤖 6. Automation & Programmability - SDN, APIs ▼
! Activation des APIs (IOS XE)
ip http secure-server ! Active HTTPS (requis pour RESTCONF)
restconf ! Active l'API RESTCONF
netconf-yang ! Active l'API NETCONF (SSH port 830)
! Configuration management
show running-config | section ospf ! Filtre la config
show running-config | include ip address ! Recherche dans config
show running-config | exclude shutdown ! Exclut les lignes
show running-config | begin interface ! Commence à une ligne
! SDN et Controller-based networking
show platform software yang-management process ! Vérifie YANG/NETCONF
show access-lists ! Voir les compteurs "hits" des ACL
! JSON, XML, YAML - Exemples de structure
! JSON:
{ "interface": { "name": "Gi1", "ip": "192.168.1.1" } }
! YAML:
interface:
  name: GigabitEthernet1
  ip: 192.168.1.1
! XML:
<interface><name>Gi1</name><ip>192.168.1.1</ip></interface>
! Outils d'automatisation (concepts)
! Ansible: utilise YAML pour les playbooks
! Puppet: utilise des manifests Ruby
! Chef: utilise des recipes Ruby
! REST API: HTTP methods (GET, POST, PUT, DELETE)

💡 Astuces importantes pour l'examen :

  • ? : Aide contextuelle - tapez ? à tout moment
  • Tab : Auto-complétion - tapez conf + Tab = configure
  • show run | section : Filtre la configuration (ex: show run | section ospf)
  • Wildcard mask : Inverse du subnet mask (0.0.0.255 = /24)
  • Distance Administrative : Connected=0, Static=1, OSPF=110, RIP=120
